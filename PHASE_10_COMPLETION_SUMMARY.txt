================================================================================
TECHNICAL DECISIONS
================================================================================

1. No External Libraries
   ✅ Browser's native print-to-PDF instead of html2pdf
   ✅ Pure CSS for layout
   ✅ Vanilla JS for calculations

2. Computed Properties Architecture
   ✅ Single reportMetrics property recalculates on date change
   ✅ Reactive system handles updates automatically
   ✅ No separate API calls needed

3. Tax Tagging Simplicity
   ✅ isTaxRelevant boolean flag (not complex mapping)
   ✅ User has full control
   ✅ Filters dynamically

4. Export Flexibility
   ✅ 4 formats: JSON (summary), JSON (full), CSV, Print
   ✅ Each format for different use case
   ✅ All formats easily extendable

5. Dark Sigma Theme
   ✅ Consistent color variables throughout
   ✅ Blue accent (#60a5fa) for active states
   ✅ Green/Red for financial positive/negative
   ✅ Print styling with black-on-white
