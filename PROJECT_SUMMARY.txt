FEATURES IMPLEMENTED (MVP v1.0)
===================================================================

BUDGET TRACKING:
  - Multi-account (checking, savings, credit card)
  - Transaction CRUD (create, read, update, delete)
  - Date-grouped transaction history with search/filter
  - 10 pre-built categories (groceries, dining, gas, etc)
  - Envelope budgeting (monthly limits per category)
  - Budget overspend warnings (color-coded)
  - Net worth calculation (all accounts)
  - Savings goals with SVG progress rings
  - Monthly dashboard with summary cards
  - Category spending breakdown charts

TRANSACTION INPUT (3 Methods):
  - Manual form (amount, category, date, description)
  - Receipt OCR (take photo > AI parses everything)
  - Natural language ("spent $45 on groceries")

AI FEATURES (Optional):
  - LLM endpoint configuration (local or cloud)
  - Vision endpoint (separate, for receipt OCR)
  - Streaming support (SSE) with single-shot fallback
  - Receipt image compression (client-side)
  - Smart category suggestion (merchant name map)
  - Connection testing (both endpoints)

DATA MANAGEMENT:
  - localStorage persistence (accounts, transactions, budgets, goals)
  - IndexedDB for receipt images (separate quota)
  - Export data as JSON
  - Import data from JSON
  - Clear all data with confirmation
  - Auto-save on changes (debounced)

PWA & OFFLINE:
  - Service Worker (offline caching)
  - manifest.json (install to home screen)
  - Works 100% without internet (after first load)
  - All data stays on device (localStorage)
  - No cloud required, no sync needed

UI/UX:
  - 5-tab bottom navigation

SETTINGS:
  - Currency selection (USD, EUR, GBP, JPY)
  - Theme toggle (dark/light)
  - LLM endpoint configuration
  - Vision endpoint configuration
  - API key management
  - Connection testing
  - Export/import data

MOBILE & PWA:

DEPLOYMENT:
